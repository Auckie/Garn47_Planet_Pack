# Garn47 Planet Pack
Adds various celestial bodies from hit game Garn47 along with a couple partially original planets

## Celestials

### Kerbol System
- Car Planet
- Carr Planet

### Garnutopiacopia System
- Bakewave
- Abliss
- Marigid
- Earth

## Mod Dependencies
- Kopernicus

## Mod Reccomendations
- Config KSP Music https://github.com/Chitak985/config-ksp-music/tree/main
- Scatterer

## Current Issues
- The temperature curves are fucked up, so you'll want to enable "Ignore Max Temperature" if you plan on visiting Garnutopiacopia

## Install Instructions
1. Click the green 'Code' button
2. In the drop-down menu, click 'Download ZIP'
3. Once the ZIP has been downloaded, copy the 'Garn' folder and drag it into your GameData folder
